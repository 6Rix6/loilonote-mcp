# ロイロノート MCP

## Requirement

- nodejs
- pnpm
- (chrome)

## Getting Started

```sh
# このリポジトリをクローンもしくはダウンロードして移動
git clone https://github.com/6Rix6/loilonote-mcp.git
cd loilonote-mcp

# 依存関係のインストールとビルド
pnpm install
pnpm run build
```

ビルド後 chrome の`パッケージ化されていない拡張機能`から`package/extension/.output/chrome-mv3`を読み込み  
MCP クライアントは Claude Desktop などお好みで
